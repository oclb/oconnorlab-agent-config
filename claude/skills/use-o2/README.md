# O2 SLURM Reference Playbook

Reference guidance for O2 partition selection, resource estimation, SLURM script templates, monitoring, and troubleshooting.

## How to use
- Use after `../init-project/subskills/setup-o2/SKILL.md` has configured bridge access.
- Use directly when operating the bridge, submitting jobs, or drafting and reviewing job scripts.

## Contents
- Partition guidance (CPU, GPU, high-memory)
- Time/memory/cpu/gpu sizing heuristics
- Reusable SLURM script templates
- Common failure modes and fixes
- Quick command reference
