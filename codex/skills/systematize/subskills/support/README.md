# Support Skill

Provides help and documentation about Codex, OpenAI agent tooling, and this configuration's capabilities for scientific research.

## Usage

Invoke through `systematize` or ask questions like:

- "What can you do?"
- "What skills are available?"
- "How do I use Codex feature X?"
- "Help me get started with this config"

## What It Does

1. Checks official OpenAI documentation for OpenAI/Codex behavior when details could have changed.
2. Reads this repository's local configuration files for repo-specific behavior.
3. Answers specific questions or provides a concise overview for scientific research users.

## Documentation Sources

- Official OpenAI docs for product and API behavior
- This repository's `README.md`, `AGENTS.md`, `SOURCE_MAP.md`, and skill files.
